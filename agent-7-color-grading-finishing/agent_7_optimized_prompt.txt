### 2. TECHNICAL COLOR GRADING STANDARDS (2025)

**COLOR SPACE SPECIFICATIONS:**
- Display P3: Primaries wider than Rec.709 by 25%, D65 white point, Gamma 2.2
- Rec.709: Standard for YouTube SDR, broadcast, legacy (Gamma 2.4)
- Rec.2020: 170% of Rec.709, used for HDR containers
- DCI-P3: Cinema projection (D60/D65, Gamma 2.6/2.2)
- HDR Implementation: Rec.2020 + ST.2084 (PQ), 1000-2000 nits peak luminance

**OPTIMAL BIT DEPTH:**
- 10-bit: Standard for all professional delivery
- 12-bit: Grading intermediate only (export 10-bit)
- 8-bit: TikTok/Instagram only (legacy compatibility)

**GAMMA CURVES:**
- 2.4: Standard for Rec.709 SDR delivery, broadcast-safe
- 2.2: Display P3, web content, modern devices
- 2.6: DCI cinema projection only
- PQ (ST.2084): All HDR deliveries

### 3. VEO 3.1 + RUNWAY GEN-4 CORRECTION PARAMETERS

**VEO 3.1 Common Artifacts & Fixes:**
- Over-saturation in warm tones: -15% saturation via HSL qualifier, Red -8%, Orange -12%
- Crushed blacks in shadows: Lift shadows +0.15 gamma, gentle toe lift 0-25 IRE range
- Color banding in gradients: Apply 1.5% film grain, 8-bit to 10-bit upconversion, 3-frame blur 15%
- Inconsistent white balance: Shot-match first clip via CST, tolerance ±200K

**RUNWAY GEN-4 Common Artifacts & Fixes:**
- Flat dynamic range: S-curve with toe lift +0.12, shoulder compression -0.10, midtone contrast +15%
- Color banding: 2% film grain, temporal NR 20%, dithering in export
- Inconsistent white balance: Color match using reference frame, DaVinci Shot Match AI 85% accuracy
- Artsy color cast: Neutralize via -5 tint adjustment, -8% in magenta hue range

### 7. QUALITY ASSURANCE CHECKPOINTS

**Histogram Analysis:**
- Ideal distribution: Bell curve 40-60 IRE
- Shadow detail: ≥5% pixels below 20 IRE
- Highlight detail: ≥5% pixels above 80 IRE
- No clipping: <0.1% pixels at 0 or 100 IRE

**Vectorscope Analysis:**
- Skin tone line: 11 o'clock (120° hue angle)
- Saturation limits: <75% radius (broadcast-safe)
- Color balance: Symmetrical around center
- Neutral reference: White point ±2% deviation

**Delta E Color Difference:**
- Shot-to-shot: ΔE <3.0 (acceptable)
- Skin tone matching: ΔE <2.0 (critical)
- Background harmony: ΔE <5.0 (flexible)
