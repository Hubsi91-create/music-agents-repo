# Agent 5a: Veo 3.1 Prompt Generator

## Übersicht
Spezialisierter Agent für production-ready Google Veo 3.1 Video-Prompts mit nativem Audio-Design und 4K-Qualität.

## Expertise
- Veo 3.1 Best Practices (Nov 2025)
- 6-teilige Goldene Formel
- Shot-by-Shot Storytelling
- Natives Audio-Design (SFX, Musik, Dialoge)
- Ingredients to Video (Character Consistency)
- Timestamp Prompting
- 4K UHD Generierung

## Input
- Video Concept (Agent 3)
- Screenplay (Agent 4)
- Music Genre & Mood
- Character Reference Images

## Output
JSON mit:
- Veo 3.1 optimierte Prompts
- Shot-by-Shot Planung
- Audio-Design Spezifikationen
- Seed-Werte für Konsistenz
- Ingredients-to-Video Empfehlungen

## Goldene Formel
[Kameraarbeit] + [Subjekt & Aussehen] + [Aktion & Bewegung] + [Szenerie & Umgebung] + [Stil, Licht & Stimmung] + [Audio-Design]

## Technische Specs
- Resolution: 4K UHD native
- Duration: 60 Sekunden (Scene Extension möglich)
- Aspect Ratios: 16:9, 9:16, 21:9, 1:1
- Audio: Native Text-to-Audio (SFX, Musik, Dialoge)
- Watermark: SynthID (unsichtbar)

## Key Features
- Shot-by-Shot Consistency via Seeds
- Ingredients to Video für Characters
- Timestamp Prompting für komplexe Sequenzen
- Negative Prompts für Artifact-Prevention
- Front-Loading für Priorität
